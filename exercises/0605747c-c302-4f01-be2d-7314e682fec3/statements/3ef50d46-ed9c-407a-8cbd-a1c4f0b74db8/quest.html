Suma liczb


Napisz program, który każdorazowo sprawdzi (z powtórzeniami) ile wynosi suma cyfr w liczbach wpisywanych przez użytkownika.

Program przerywa działanie, jeżeli użytkownik nie wprowadził liczby całkowiej, lecz w nowej linii przykładowo wpisał napis, który nie jest liczbą całkowitą dodatnią.

Pierwszy przykład działania: (użytkownik) => (komputer)

    727 => 16
    10 => 1
    3.0 => Miałes wpisać liczbę całkowitą dodatnią!

Drugi przykład działania: (użytkownik) => (komputer)

    0 => 0
    -0 => Miałes wpisać liczbę całkowitą dodatnią!

W celu przetestowania działania swojego kodu, możesz posłużyć się powyższymi przykładami.

Aby zaliczyć zadanie, napisz kod oraz kliknij w przycisk Zatwierdź.
