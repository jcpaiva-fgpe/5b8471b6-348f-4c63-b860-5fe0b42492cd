Dominant


Write a program that reads a sequence of natural numbers separated by spaces, and returns the z them which occurs the greatest number of times. If more than one number occurs so many same (most) times, just return the first one.


Examples of operation:

The following form of interpretation (user) => (computer)

    3 12 3 4 => 3
    7 4 1 1 7 1 => 1
    2 3 => 2

The program does not have to display the number of times the numbers are repeated, it only has to display the number witch repetes in the sequence at most (dominates).
