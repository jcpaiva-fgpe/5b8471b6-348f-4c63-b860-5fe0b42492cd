Takie same wyrazy


Napisz program, który wczyta od użytkownika dwa napisy i wyświetli tylko takie wyrazy, które występują w obu tych napisach naraz. Można użyć metody split napisu.

Przykład przebiegu wykonania:

    Komputer: Podaj pierwszy napis:

    Użytkownik: Ala ma kota

    Komputer: Podaj drugi napis:

    Użytkownik: Ala ma psa

    Komputer: Takie same wyrazy to: Ala

W celu przetestowania działania swojego kodu, możesz posłużyć się powyższym przykładem.

Aby zaliczyć zadanie, napisz swój kod oraz kliknij w przycisk Zatwierdź.
