Palindromy


Napisz program, który sprawdzi, czy słowo podane przez użytkownika jest palindromem, tzn. od tyłu czyta się tak samo jak od przodu.


Przykłady działania. Poniższa forma zapisu (użytkownik) => (komputer)

    radar => tak

    bulba => nie

Wielkość litery nie może mieć znaczenia.


Aby zaliczyć zadanie, napisz swój kod oraz uruchom go podając następujące dane wejściowe: Ala.
